# English Quizzes

Advanced-level interactive quizzes for upper intermediate to advanced English learners.

## Quizzes

1. **Cancel Culture Paradox** - Explore accountability vs. mob rule with advanced conditionals and inversion
2. **Technology & Mental Health** - Analyze social media's impact on youth using passive voice and reporting verbs
3. **Gene Editing Ethics** - Debate CRISPR and designer babies with future forms for speculation

## Features

- 15 questions per quiz
- Multiple choice with randomized alternatives
- Immediate feedback
- Performance breakdown by category
- Brutalista, minimalist design
- No data collection or tracking

## How to Use

Visit the site and select a quiz. Answer all 15 questions and submit to see your results.

## Development

To add a new quiz:
1. Create the quiz HTML file
2. Add it to the index.html with a quiz card
3. Commit and push to GitHub
